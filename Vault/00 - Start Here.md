# 🗳️ CrowdVote AI: The Electoral OS

Welcome to the **CrowdVote AI** (Electoral OS) Documentation Vault. This project is a high-fidelity, real-time election prediction and tracking platform specifically tailored for the **Kerala 2026 Assembly Elections**.

## 🎯 Project Goal
To harness the "Wisdom of the Swarm" to predict electoral outcomes with high precision, using a combination of historical data analysis and real-time user-driven consensus.

## 🛠️ Tech Stack
- **Frontend**: React 19 + Vite + Tailwind CSS 4
- **Backend**: Node.js + Express
- **Database**: MongoDB (Atlas)
- **Real-time**: Socket.io
- **Security**: Firebase Authentication (Email/Google/Phone OTP)
- **Design**: Lucide Icons + Framer Motion (Animations)
